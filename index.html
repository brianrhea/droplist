Droplist

a jQuery plugin for breadcrumbs, directory trees,
or whatever else you may think of.

Download from GitHub

Overview

Often, to preserve vertical space, you want your breadcrumbs to be on one line. But, what if your design dictates that you need the breadcrumb in a narrow column and you're short on horizontal space?

Droplist allows you to hide the overflowing crumbs and then makes them available to your visitor via a dropdown menu.

Default

$( ".droplist" ).droplist();

Overflow on the Right

If you'd rather hide the elements at the end of your list; just pass the overflow option with a value of right.

  • Home
  • Tech
  • Apps & Software
  • What's New
  • Knock to Unlock
$( ".droplist" ).droplist({
  overflow: 'right'
});

Dropicon

But most likely, that boring caret is getting on your nerves and you want to override it. If you're using the Font Awesome icon library, it's as simple passing in the icon class of your choice like so:
