Droplist

a jQuery plugin for breadcrumbs, directory trees,
or whatever else you may think of.

Download from GitHub

Overview

Often, to preserve vertical space, you want your breadcrumbs to be on one line. But, what if your design dictates that you need the breadcrumb in a narrow column and you're short on horizontal space?

Droplist allows you to hide the overflowing crumbs and then makes them available via a dropdown menu which you can customize.
